Задание:
Вы тестируете систему бронирования, включающую операции создания, подтверждения и отмены бронирований.

Ваша цель — организовать тесты с помощью аннотации @Nested, чтобы логически сгруппировать их для различных состояний
бронирования. Каждый уровень вложенности требует настройки инициализации и очистки для корректного управления ресурсами, связанными с состояниями.

Основной уровень Настройте BookingService, добавив инициализацию и завершение работы сервиса в методах initializeService и logOnServiceCleanUp.

Вложенные классы и логика:В классе WhenBookingIsCreated добавьте логику для создания объекта Booking, настроив его в initializeBooking,
настройте завершение работы с этим объектом в методе logOnTheCleanup.В WhenBookingIsConfirmed обеспечьте инициализацию состояния бронирования в методе confirmBooking
и настройте завершение в cleanupConfirmedBooking.

В классе WhenBookingIsCreated добавьте логику для создания объекта Booking, настроив его в initializeBooking, настройте завершение работы с этим объектом в методе logOnTheCleanup.

В WhenBookingIsConfirmed обеспечьте инициализацию состояния бронирования в методе confirmBooking и настройте завершение в cleanupConfirmedBooking.

Методы тестов Добавьте тестовые методы, которые проверят корректность состояния бронирования на каждом этапе: для создания, подтверждения и отмены бронирования.
