Вы проверяете систему с динамически создаваемыми тестами на основе файлов, находящихся в директории src/test/resources/urls.
Каждый файл в этой папке содержит строку с URL-адресом. Набор файлов может меняться, и вам нужно проверить, что каждый URL в файлах начинается с "http".
Создайте динамический тест для каждого файла, проверяя, что URL начинается с "http", и выводите в консоль название теста и проверяемый URL.
